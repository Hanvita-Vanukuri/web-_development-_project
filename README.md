# Web Development Projects Collection

This repository contains 5 interactive web development projects built using HTML, CSS, and JavaScript. These projects demonstrate concepts of game development, data visualization, algorithm visualization, and interactive UI design.

---

# Projects Included

## 1. Sudoku Puzzle Generator
An interactive Sudoku Puzzle Game where users can solve randomly generated Sudoku boards directly in the browser.

### Features
- Interactive 9x9 Sudoku board
- Random puzzle generation
- Input validation
- Responsive design
- Restart and new game functionality

### Technologies Used
- HTML5
- CSS3
- JavaScript

---

## 2. Snake Game
A classic Snake Game developed using JavaScript and HTML5 Canvas.

### Features
- Real-time snake movement
- Keyboard controls
- Score tracking
- Collision detection
- Game over screen

### Technologies Used
- HTML5 Canvas
- CSS3
- JavaScript

---

## 3. Sales Dashboard with Charts
An interactive dashboard that visualizes sales data using multiple chart types.

### Features
- Bar Chart
- Line Chart
- Pie Chart
- Doughnut Chart
- Radar Chart
- Responsive dashboard layout

### Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js

---

## 4. Interactive World Map with Population Census
A world map visualization project displaying country-wise population census data interactively.

### Features
- Interactive world map
- Country hover effects
- Population tooltips
- Zoom and pan support
- Responsive design

### Technologies Used
- HTML5
- CSS3
- JavaScript
- SVG / GeoJSON

---

## 5. QuickSort Algorithm Visualization
A visual representation of the QuickSort algorithm using animated bars and sorting steps.

### Features
- Real-time sorting visualization
- Adjustable animation speed
- Random array generation
- Step-by-step sorting process
- Interactive controls

### Technologies Used
- HTML5
- CSS3
- JavaScript
